自介與動機

我叫廖浩君，曾就讀於台南大學視覺藝術與設計學系，擁有設計背景及現場解決問題的經驗。
2021/5-2022/6在大聲設計擔任視覺設計師一年，客製化設計品牌視覺系統並優化公司網站，使客戶轉換率提升。
後在大塚製藥2023/4-2024/9擔任現場作業員，著手排除機台問題並提高生產效率。
因緣際會下在選擇具有專業性的職涯時，發掘前端工程師是一個充滿挑戰且具發展潛力的領域，決心轉職並已全力投入學習，期待參加WeHelp訓練營，磨練實力並進入前端職場。

學習經驗

  1. 網站優化經驗：曾使用套版網站優化公司頁面，透過視覺調整和內容配置，成功提升網站轉換率。
  2. WebFlow 自建網站：嘗試個人網站架設。
  3. HTML / CSS 基礎學習：透過Udemy線上課程，打下前端基本功。
  4. JavaScript 基礎學習：在BiliBili平台學習黑馬程序員課程，進行習題練習，加強邏輯思維與應用能力。

學習安排

目前已辭去工作，擁有全職學習的時間，將每天投入8-10小時進行WeHelp訓練營的排程和任務。
每週一早上 10:00 ：任務安排與引導關鍵技術。

    根據任務去找資源，並分析所需要的總時數，再由時數去切成五等份。
  • 上午：根據每日切分出來的任務去實作，並將不懂與疑惑的點先記錄起來。
  • 下午：重複上午的內容，在休息前開始找答案。
  • 晚上：撰寫每日進度報告。

投入領域與回報

  1. 市場需求：透過Google趨勢及104職缺分析，發現前端工程師在市場上有穩定的需求。
  2. 個人期待：我對創造具吸引力的互動體驗充滿熱情。過去在視覺設計工作中，我學會了如何將抽象概念轉化為具體的視覺系統，現在希望透過程式開發，將這些設計帶入網站或應用中，實現更豐富的用戶體驗與更高的薪資。

情緒管理

經歷：在設計公司工作時，因為外包案件，導致品質下滑，也造成高壓的重工情況。
處理方式：我選擇冷靜面對，先將當下工作完成，先確保客戶滿意。隨後反思這份工作與自己的初衷不符，決定離職並重新尋找能發揮專長的方向。這次經驗也讓我學會在負面情緒中保持理智，專注於解決問題。

技術心得

過去在進行網站優化時，體會到SEO與畫面視覺的重要性。調整版面配置與視覺設計可以提升使用者停留時間，在相同預算廣告費下也更能有效提升SEO及整體轉換率。這讓我理解技術與設計必須協同合作，才能達到最佳效果。

職涯與社會

我計劃成為一名前端工程師，結合過去的視覺設計背景與新學到的開發技能，專注於打造視覺和使用體驗優秀的網頁或應用程式。我預計在 6 月中求職，希望能加入一個鼓勵成長的團隊，持續學習並產出優質的產品。

其他事項

從設計師到現場作業員的轉換，再到現在全心學習前端技術，我始終保持學習的熱忱與解決問題的耐心。過程中雖曾碰壁，但每次挫折都讓我更清楚自己的方向。參加WeHelp Bootcamp是我轉職路上的關鍵一步，期待透過嚴格的訓練，蛻變成一名真正能為團隊帶來價值的前端工程師。
